CADEIA 1 — ENTRADA E MOTOR CENTRAL
------------------------------------------------------------------------------
01_INPUT / tabela INPUT -> 03_TB_HCI
03_TB_HCI calcula TOTAL_SERIE, RESULTADO_EVENTO, médias, sequências, scores HCI e trava FINAL.
Trava FINAL esperada na TB_HCI: HCI_SERIE_ORDER, HCI_FINAL_SERIE_ORDER, HCI_EVENT_ROW_VALID.

CADEIA 2 — GRÁFICOS E DASHBOARD
------------------------------------------------------------------------------
03_TB_HCI -> 07_HCI_PERF_TARGETS -> CHART_TARGETS_OUTPUT
03_TB_HCI -> 08_HCI_PERF_STRUCTURE -> CHART_STRUCTURE_OUTPUT
03_TB_HCI -> 05_HCI_RHYTHM_TIMELINE -> 06_HCI_RHYTHM_PATH -> CHART_RHYTHM_OUTPUT
Os gráficos na OUTPUT são copiados por VBA para APP_DASHBOARD_VBA.

CADEIA 3 — DIAGNÓSTICO, DECISÃO E OUTPUT
------------------------------------------------------------------------------
03_TB_HCI -> 09_HCI_KNOWLEDGE_PLAN -> 10_TB_HCI_OUTPUT_DECISION_ENGINE
10_TB_HCI_OUTPUT_DECISION_ENGINE -> 12_HCI_OUTPUT_QUALITATIVE_INTELLIGENCE -> OUTPUT/RELATORIO
OUTPUT deve ser camada de leitura; decisões ficam nos motores.

CADEIA 4 — TREINAMENTO TÉCNICO
------------------------------------------------------------------------------
10_TB_HCI_OUTPUT_DECISION_ENGINE + 16_TB_HCI_TRAINING_BI_CLASSIFIED/config/contexto/fase -> 14_TB_HCI_TRAINING_ENGINE_FINAL
14_TB_HCI_TRAINING_ENGINE_FINAL -> 15_TB_HCI_TRAINING_CONSULTATION -> OUTPUT/RELATORIO
Regra aprovada: Consultation não deve exibir treino físico nem mental puro.

CADEIA 5 — TREINAMENTO FÍSICO
------------------------------------------------------------------------------
19_TB_HCI_ATHLETE_PROFILE_ENGINE -> 20_TB_HCI_PHYSICAL_TRAINING_BI -> 21_TB_HCI_PHYSICAL_TRAINING_PACKAGES -> 22_TB_HCI_PHYSICAL_TRAINING_PACKAGE_OUTPUT
22_TB_HCI_PHYSICAL_TRAINING_PACKAGE_OUTPUT alimenta card físico da OUTPUT.
