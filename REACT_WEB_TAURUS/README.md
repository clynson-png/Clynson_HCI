# REACT_WEB_TAURUS

Projeto React separado do `HCI_REACT_WEB`, criado a partir da base já estruturada do app existente.

Objetivo:
- manter uma frente própria para TAURUS
- reaproveitar as lógicas já estruturadas de `src`
- evoluir o produto sem acoplar novamente à frente antiga

Origem da cópia:
- `C:\HCI_REACT_WEB\02_REACT_WEB\hci-react-web`

Escopo copiado:
- `src`
- `public`
- arquivos de build e configuração do Vite
- proposta visual `taurus-target-proposal.html`

Escopo não copiado:
- `node_modules`
- `dist`

Comandos:

```bash
npm install
npm run build
npm run dev
```
